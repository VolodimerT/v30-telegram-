# ⚡ QUICK START - ЗАПУСК ЗА 5 МИНУТ

## 🚀 快速安装 (Fast Setup)

### Шаг 1️⃣: Подготовка (1 минута)
```bash
# Скачать проект
mkdir autonomous_bot && cd autonomous_bot

# Виртуальное окружение
python3 -m venv venv
source venv/bin/activate  # или venv\Scripts\activate на Windows

# Зависимости
pip install -r requirements.txt
```

### Шаг 2️⃣: Конфигурация (2 минуты)
```bash
# Создать .env файл
cat > .env << EOF
TELEGRAM_BOT_TOKEN=YOUR_BOT_TOKEN_FROM_BOTFATHER
ODDS_API_KEY=bafb678b8ac2d2ee7cd88fdc9318d308
PERPLEXITY_API_KEY=optional_if_you_have
EOF
```

### Шаг 3️⃣: Запуск (1 минута)
```bash
python main.py
```

✅ **БОТ РАБОТАЕТ!**

## 📱 Первые шаги в Telegram

### 1. Проверить статус
Напишите боту:
```
/start
```

Должно быть:
```
🎯 AUTONOMOUS BETTING BOT v2.0
💰 Bank: 1019
🚀 Status: AUTONOMOUS RUNNING
```

### 2. Ждите первого скана
Бот сканирует **каждый час** по расписанию:
- `:00` часа (12:00, 13:00, 14:00...)

### 3. Одобрить ставку
Когда придёт кандидат:
```
/pending
```

Затем:
```
/approve event_id
```

## ⚠️ Важно!

### 🔴 НИКОГДА не включайте `enable_auto_place`!
```yaml
# ❌ НЕТ! (в bot_config.yaml)
apis:
  bookmakers:
    enable_auto_place: true  # ← ОПАСНО!
```

Это может автоматически разместить РЕАЛЬНЫЕ деньги!

### 🟡 НАЧНИТЕ с EMERGENCY режима
```yaml
# ✅ ДА! (по умолчанию)
bank:
  current_mode: EMERGENCY
```

Это самый консервативный режим:
- EV >= 8.0%
- Максимум 2 ставки в день
- Макс ставка = 2% баланса

### 🟢 МОНИТОРЬТЕ drawdown
Если потери > 20% → бот автоматически переходит в FREEZE режим

## 🆘 Что-то не работает?

### ❌ Бот не запускается
```bash
# Проверьте:
python --version  # >= 3.9?
echo $TELEGRAM_BOT_TOKEN  # Установлен?
cat .env | head -3  # Файл существует?
```

### ❌ Нет кандидатов на ставки
```bash
# Проверьте логи:
tail -f logs/scanner.log

# Вероятные причины:
# 1. Очень сложная фильтрация (EV 8%)
# 2. Нет живых матчей в спортах из конфига
# 3. Odds API не работает
```

### ❌ Perplexity ошибка
```
PERPLEXITY_API_KEY не установлен
Решение: добавить в .env или оставить пусто (анализ отключится)
```

### ❌ Telegram не приходят уведомления
```bash
# Проверьте:
1. Отправили ли сообщение боту в Telegram?
2. /start команда работает?
3. Логи: tail -f logs/errors.log
```

## 📈 Как фильтры работают

### ФИЛЬТР 1: EV (Expected Value)
```
Odds = 2.10 (Arsenal)
Вероятность = 55% (из анализа)

EV = (0.55 × 2.10) - 1 = 0.155 = 15.5%

✅ 15.5% >= 8.0% (минимум) → PASS
```

### ФИЛЬТР 2: Коэффициенты
```
Min odds: 1.5
Max odds: 5.0

2.10 находится в диапазоне → PASS
```

### ФИЛЬТР 3: Количество букмекеров
```
Нужно минимум 3 букмекера с таким коэффициентом

Pinnacle: 2.10 ✅
Bet365: 2.09 ✅
Betfair: 2.08 ✅ (всего 3)

→ PASS
```

### ФИЛЬТР 4: RED FLAGS
```
Проверяем анализ на:
- "договор" / "fixing" → ❌ STOP
- Дисквалификация ключевого игрока → ❌ STOP
- "scandal" → ❌ STOP

Если нет → PASS
```
